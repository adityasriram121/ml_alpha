If your goal is **"learn the most ML from one project"**, I would not start with neural networks.

## Project: Predict Exoplanet Properties

Since you're already working with the NASA exoplanet dataset, use that.

### Goal

Predict a planet's mass from other features.

Input features:

* Radius
* Orbital period
* Distance from Earth
* Discovery method
* Discovery year

Target:

* Planet mass

### What you'll learn

#### 1. Data Cleaning

* Missing values
* Dropping bad rows
* Feature selection
* Data quality

#### 2. Pandas

* Filtering
* Grouping
* Aggregations
* Feature engineering

#### 3. Visualization

* Histograms
* Scatter plots
* Correlation heatmaps

#### 4. Classical ML

Train multiple models:

1. Linear Regression
2. Decision Tree
3. Random Forest

You'll learn:

* Training
* Testing
* Predictions
* Overfitting
* Underfitting

#### 5. Evaluation

Learn:

* MAE
* MSE
* RMSE
* R² Score

These are fundamental ML metrics.

#### 6. Feature Engineering

Create new features:

```python
mass_radius_ratio
log_period
planet_density_estimate
```

You'll discover how features affect performance.

#### 7. Model Comparison

Build a table:

| Model             | R² |
| ----------------- | -- |
| Linear Regression | ?  |
| Decision Tree     | ?  |
| Random Forest     | ?  |

Then explain why one wins.

## Stretch Goal

Predict planet type:

Classes:

* Rocky
* Super-Earth
* Neptune-like
* Gas Giant

Now you learn:

* Classification
* Accuracy
* Confusion matrices
* Precision
* Recall

## Skills Gained

By the end you'll understand:

* Pandas
* Data cleaning
* Visualization
* Feature engineering
* Regression
* Classification
* Train/test split
* Overfitting
* Model evaluation
* Scikit-learn workflow

That's roughly **70–80% of what a beginner ML engineer uses daily**.

For someone starting from zero and already interested in exoplanets, this is one of the highest learning-per-hour projects you can do.
